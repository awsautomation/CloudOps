<Intro>

[Pulumi](https://www.pulumi.com/) is an open-source infrastructure-as-code tool that lets you use general-purpose programming languages like TypeScript, Python, Go, and others to build, deploy, and manage cloud infrastructure. You can use Pulumi to manage resources from many popular cloud providers, including New Relic.

</Intro>
